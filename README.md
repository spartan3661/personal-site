# Personal Portfolio Website

A cyberpunk-themed personal portfolio website for Ben Carnes, built to showcase software development experience, projects, technical skills, and contact information.

## Features

- Animated matrix rain background
- Neon cyberpunk visual theme
- Custom cursor and scanline effects
- Interactive project cards
- Retro terminal modal
- Responsive layout
- Contact form integration
- Resume download link

## Tech Stack

- HTML
- CSS
- JavaScript
- Formspree for contact submissions

## Sections

- Hero introduction
- Work history
- Projects
- Skills
- Contact form
- Interactive terminal

## Highlighted Projects

- **Gamegrab** — Desktop OCR and translation tool for PC games using PySide6, EasyOCR, PyTorch, C++, and DeepL API.
- **WGCapture** — Python screen-capture library on PyPI backed by C++, DLLs, Direct3D, and Windows APIs.
